# Analytics-Dashboard-📊 

A responsive **Analytics Dashboard** built with **HTML, CSS, and JavaScript**.  
It visualizes repository activity such as commits, issues, and pull requests using **Chart.js**.

---

## 🚀 Features
- Sidebar navigation for quick access to sections
- Summary cards showing:
  - Total Commits
  - Open Issues
  - Pull Requests
- Interactive line chart for weekly activity
- Responsive design for desktop and mobile
- Easy to extend with more charts or tables

---

## 🛠️ Tech Stack
- **HTML5** – structure
- **CSS3** – styling
- **JavaScript (ES6)** – interactivity
- **Chart.js** – data visualization
